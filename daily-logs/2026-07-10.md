## What I Learned

- A Region is a geographic area containing AWS infrastructure
- An Availability Zone is an isolated location within a Region
- AWS is responsible for security of the cloud
- Customers are responsible for security in the cloud
- IAM controls who can access AWS and what actions they can perform
- IAM roles provide temporary permissions
- Least privilege means granting only the permissions required
- EC2 provides virtual servers
- S3 stores objects and files
- Lambda runs code without managing servers
- A VPC is a logically isolated network in AWS
- A public subnet has a route to an Internet Gateway
- Security groups operate at the resource level
- Network ACLs operate at the subnet level
- Git status and Git log help track repository changes
- Mermaid can be used to create architecture diagrams in Markdown
