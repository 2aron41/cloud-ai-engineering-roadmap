## What I Learned

- Edge locations bring cached content closer to users.
- Private subnets do not have a direct route to the internet through an Internet Gateway.
- S3 Standard is for active files that need fast access.
- S3 Glacier is for archived files with slower retrieval.
- Route tables decide where network traffic goes.
- A public subnet needs a route like `0.0.0.0/0 -> Internet Gateway`.
- A NAT Gateway lets private subnet resources access the internet outbound without being directly reachable from the internet.
- Security Groups protect resources and are stateful.
- NACLs protect subnets and are stateless.
- CloudWatch is for metrics, logs, alarms, and monitoring.
- CloudTrail records AWS API activity and account actions.
- S3 stores the portfolio files.
- CloudFront delivers the portfolio faster and more securely.
- `grep` searches text for matching patterns.
- `find` searches for files and folders.
- `tar` creates archive files and can compress multiple files into one package.
