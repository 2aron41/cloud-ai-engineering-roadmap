# AWS Mini Quiz — July 13, 2026

## 1. Why should active portfolio files use S3 Standard instead of S3 Glacier?

Answer: Active files require immediate, low-latency access and are retrieved frequently

## 2. What does CloudFront do that S3 alone does not?

Answer:CloudFront is a globally distributed Content Delivery Network

## 3. What does ACM provide?

Answer:provisions, manages, and automatically renews SSL/TLS certificates

## 4. Why should an S3 bucket avoid public write access?

Answer:Public write access allows anyone on the internet to upload, overwrite, or delete files in your bucket.

## 5. What is the difference between CloudWatch and CloudTrail?

Answer:CloudWatch monitors the performance and operational health of your resources. CloudTrail audits account activity by logging all API calls made by users or services

## 6. What does a NAT Gateway allow private subnet resources to do?

Answer:It allows resources in a private subnet to initiate outbound connections to the internet

## 7. What makes a subnet public?

Answer:if its route table includes a direct route to an Internet Gateway

## 8. What makes Security Groups different from NACLs?

Answer:Security Groups act at the instance level and are stateful

## 9. Why should root not be used for daily AWS work?

Answer:The root user has unrestricted access to all resources and billing

## 10. What is the safest next step before deploying AWS resources?

Answer:use Infrastructure as Code
