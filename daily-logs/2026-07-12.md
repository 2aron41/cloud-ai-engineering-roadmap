## What I Learned

- S3 is object storage.
- An S3 bucket is a container for objects.
- An S3 object is a file stored in S3 plus metadata.
- Static website hosting means serving HTML, CSS, JavaScript, images, and PDFs without backend server code.
- Public access means people on the internet can access bucket files.
- Public access can be risky if private files become exposed.
- CloudFront is AWS’s content delivery network.
- Edge locations are AWS locations closer to users.
- Caching stores temporary copies of content for faster loading.
- HTTPS encrypts the website connection.
- S3 Standard should be used for active website files.
- S3 Glacier is for archive storage, not active website files.
- In the portfolio architecture, S3 stores files and CloudFront delivers them faster and securely.
