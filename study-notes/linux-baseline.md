## Questions and Answers

### What does `pwd` show?

`pwd` displays the full path of the current working directory.

### What does `ls -la` show that `ls` does not?

`ls -la` displays hidden files and detailed information, including permissions, ownership, file size, and modification date.

### What does permission `600` mean?

Permission `600` gives the owner read and write access. The group and everyone else receive no permissions.

### What does `ps` display?

`ps` displays information about currently running processes.

### What does `df -h` measure?

`df -h` displays filesystem disk usage and available storage in human-readable units.
