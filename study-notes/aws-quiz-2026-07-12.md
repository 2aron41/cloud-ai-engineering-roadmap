## Corrections

Public access in S3 means people on the internet can access the bucket or objects.

Public access can be risky because private files could accidentally become public.

CloudFront is useful in front of S3 because it makes the site faster, more reliable, supports caching, and helps with HTTPS.

HTTPS is important because it encrypts the connection and makes the portfolio look more trusted and professional.

An S3 object is a file stored in S3 plus metadata.
