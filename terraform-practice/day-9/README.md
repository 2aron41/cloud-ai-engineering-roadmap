# Day 9 Terraform Practice

This is a harmless local Terraform configuration. It defines local values and an output but creates no AWS resources.

## Commands

- `terraform init`: initialized the working directory.
- `terraform fmt`: formatted the configuration.
- `terraform validate`: confirmed that the configuration was valid.
- `terraform plan`: previewed the output without creating infrastructure.

No `terraform apply` or `terraform destroy` command was used.
